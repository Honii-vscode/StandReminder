# Stand Reminder - PWA 久坐提醒

随机间隔提醒你站立活动，保护颈椎和腰椎健康。

## 功能特点

- ✅ **随机间隔**：工作时间 20-30分钟随机，活动时间 3-5分钟随机
- ✅ **深色原生UI**：接近 iOS / Color OS 原生风格
- ✅ **震动+声音提醒**：不同提醒次数区分类型（工作结束：3震，活动结束：2震）
- ✅ **每日统计**：记录今日完成周期和累计工作时间
- ✅ **PWA 支持**：可添加到手机桌面，离线可用
- ✅ **支持暂停/继续**：随时可以暂停，随时结束
- ✅ **Wake Lock API**：尽可能防止系统杀后台

## 访问地址

服务器地址：`http://43.156.96.119:3001`

## 在 OPPO Color OS 上安装使用

### 第一步：添加到桌面
1. 用手机 Chrome 或 自带浏览器打开 `http://43.156.96.119:3001`
2. 打开菜单（右上角三个点）
3. 选择「添加到桌面」或「安装」
4. 桌面上会出现「Stand」图标

### 第二步：设置不被杀后台（非常重要！）

1. **设置 → 电池 → 应用电量管理**
2. 找到「Stand」（就是添加到桌面的这个应用）
3. **后台管理 → 选择「允许后台活动」**
4. **开启「允许自启动」**
5. **智能省电 → 加入白名单**

不同 Color OS 版本文字可能略有不同，核心就是：允许后台运行，不要清理。

### 第三步：开始使用
1. 点击桌面上的「Stand」图标打开
2. 点击「开始提醒」
3. 允许通知权限（这样锁屏也能收到提醒）
4. 开始工作！时间到了会自动提醒你站起来活动

## 管理命令

```bash
# 查看状态
pm2 status

# 查看日志
pm2 logs stand-reminder

# 重启
pm2 restart stand-reminder

# 停止
pm2 stop stand-reminder
```

## 技术栈

- 纯 HTML/CSS/JavaScript 零依赖
- Express 静态文件服务器
- PWA + Service Worker 离线支持
- Web Vibration API 震动
- Web Audio API 提示音
- Notification API 系统通知
- Wake Lock API 防止休眠
- PM2 进程守护
